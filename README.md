# Task-2:Customer Retention & Churn Analysis 

Focuses on analyzing customer retention patterns and churn behavior using Python. The dashboard provides insights into customer retention trends, churn rates across subscription plans, support ticket activity, and payment delay distributions to help businesses improve customer loyalty and reduce churn.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dashboard Highlights
- Cohort Retention Heatmap
- Churn Rate by Subscription Type
- Support Tickets Analysis (Retained vs Churned Customers)
- Payment Delay Distribution Analysis

## Key Insights
- Analyzed customer retention trends over multiple cohorts.
- Compared churn rates across different subscription plans.
- Evaluated the relationship between support ticket volume and customer churn.
- Studied payment delays and their impact on customer retention.

## Outcome
Generated actionable insights to help businesses understand customer behavior, improve retention strategies, and minimize customer churn through data-driven decision-making.
